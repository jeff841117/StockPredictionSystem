# 目前 MVP 功能摘要

本文件整理目前已經落地的主要功能，供協作者快速查閱。

若要進行 MVP 第一版總驗收，請優先搭配：

- [mvp-v1-acceptance-summary.md](mvp-v1-acceptance-summary.md)

## 股票查詢

- 單一台股代號查詢
- 自訂開始日期與結束日期
- 顯示股票名稱、日期、開高低收、成交量
- 對無效代號、日期錯誤、查無資料、外部失敗提供專案自己的提示訊息

## 圖表與技術指標

- 收盤價走勢圖
- MA5 / MA20
- 圖表資料與表格資料使用同一查詢區間

## 收藏清單

- 新增收藏
- 查看收藏
- 移除收藏
- 避免重複加入

## 模擬交易

- 模擬 BUY
- 模擬 SELL
- 買進前檢查虛擬資金
- 賣出前檢查持股是否足夠

## 交易紀錄

- 顯示 BUY / SELL 交易
- 預設新到舊排序
- 顯示交易總金額
- 顯示 SELL 已實現損益

## 持股與損益

- 持股數量
- 平均成本
- 持股成本基礎
- 最近可取得收盤價
- 估算市值
- 未實現損益
- 累計已實現損益

## 投資組合摘要

持股頁目前顯示：

- 初始虛擬資金
- 目前可用資金
- 目前已使用資金
- 目前持股市值
- 累計已實現損益
- 整體未實現損益
- 目前總資產估值

固定計算規則：

- `總資產估值 = 可用資金 + 持股市值`

## 目前已知限制

- 不是即時報價
- 僅支援單一股票查詢
- 無多使用者隔離
- 無手續費與交易稅
- 無正式部署說明
