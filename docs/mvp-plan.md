# MVP 進度與目前範圍

本文件記錄 MVP 主線目前已完成內容、尚未完成內容與下一步方向。  
重點是反映「目前版本」，不是單純的理想拆解草案。

## MVP 目標

目前 MVP 目標是完成一個可本機操作的台股分析與模擬交易系統，至少包含：

1. 單一股票查詢
2. 歷史資料表格與走勢圖
3. MA5 / MA20
4. 收藏清單
5. 模擬 BUY / SELL
6. 交易紀錄
7. 持股總覽
8. 已實現 / 未實現損益
9. 投資組合摘要

## 已完成模組

### 1. 專案基礎骨架

- FastAPI 入口
- 模板與靜態資源結構
- `.env` 載入
- README 基本啟動說明

### 2. 股票資料查詢

- 使用 TWSE `STOCK_DAY`
- 支援單一台股代號
- 支援自訂日期區間
- 查無資料、日期錯誤、無效代號與外部失敗皆有明確提示

### 3. 歷史資料顯示與圖表

- 歷史價格表格
- 收盤價 SVG 走勢圖
- 結果頁摘要資訊

### 4. 技術指標

- MA5
- MA20
- 同步顯示於表格與走勢圖

### 5. 收藏清單

- 加入收藏
- 查看收藏
- 移除收藏
- 重複加入防呆

### 6. 模擬交易

- 模擬 BUY
- 模擬 SELL
- 虛擬資金檢查
- 持股不足檢查

### 7. 交易紀錄

- 顯示 BUY / SELL 明細
- 新到舊排序
- 顯示每筆 SELL 已實現損益

### 8. 持股與損益

- 持股總覽
- 平均成本
- 持股成本基礎
- 未實現損益
- 已實現損益
- 投資組合摘要

## 目前 MVP 仍未包含

- 使用者登入
- 多使用者資料隔離
- 即時行情
- 多股票比較
- 更多技術指標
- 完整投資組合圖表分析
- 手續費、交易稅與進階交易規則
- 正式部署到外網的完整流程

## 建議操作流程

1. 啟動 FastAPI
2. 到首頁輸入股票代號與日期區間
3. 查看歷史資料、走勢圖與 MA
4. 在結果頁加入收藏或模擬買進
5. 到交易紀錄頁查看 BUY / SELL
6. 到持股總覽頁查看持股、市值與損益摘要

## 驗證方式

目前最小自動化驗證：

```bash
python -m unittest discover -s tests -v
```

人工檢查建議：

1. 首頁查詢流程
2. 收藏新增與移除
3. BUY / SELL 交易流程
4. 交易紀錄頁
5. 持股總覽頁
